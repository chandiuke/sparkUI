# SparkUI ✦

Premium UI components ready to copy & paste. Built with Tailwind + Framer Motion.

## Quick Start

```bash
# Initialize SparkUI in your project
npx sparkui init

# Add components
npx sparkui add button
npx sparkui add accordion
npx sparkui add chip
```

## Features

- ✦ Beautiful, animated components
- ✦ Works with TypeScript & JavaScript
- ✦ Page transitions with curtain effects
- ✦ Dark/light mode support
- ✦ Fully responsive & accessible
- ✦ Zero component library dependencies
- ✦ Copy & paste ready

## Available Components

| Component | Description |
|-----------|-------------|
| `button` | Versatile button with 7 variants, 6 colors, loading states |
| `accordion` | Collapsible content with 4 variants, animations |
| `chip` | Tags, labels, status indicators with close animation |
| `input` | Text input with validation states |
| `kbd` | Keyboard key indicator |

## Manual Installation

If you prefer not to use the CLI:

1. Copy the component from `components/ui/`
2. Install dependencies: `npm install clsx framer-motion`
3. Add CSS variables to your `globals.css`

## Tech Stack

- [Next.js 15](https://nextjs.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [TypeScript](https://www.typescriptlang.org/)

## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build CLI
cd cli && npm install && npm run build
```
